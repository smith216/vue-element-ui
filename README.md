

#This demonstrats vue-cli (`vue2`+`vuex`+`axios`+`vue-router`+`element-ui`+`echarts`+`websocket`+`vue-i18n`)

  

#### install

#### clone
（`git clone`）

```shell
git clone https://github.com/hzlshen/vue-project.git
```

#### install packages

```
npm install
```

#### launcg page(http://localhost:8080)

```
npm run dev
```

#### build deliveries
```
npm run build
```
